# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the MISEIA 1-1-100 web-documents project. ADRs document significant technical decisions, the context that motivated them, and the trade-offs accepted.

| ADR | Title | Status |
|-----|-------|--------|
| [ADR-001](ADR-001-mongodb-over-postgresql.md) | MongoDB over PostgreSQL for document metadata | Accepted |
| [ADR-002](ADR-002-rustfs-s3-storage.md) | RustFS (S3-compatible) for file storage | Accepted |
| [ADR-003](ADR-003-hmac-cookies-over-jwt.md) | HMAC-SHA256 HTTP-only cookies over JWT | Accepted |
